Ordboble visual reference 🌈☁️✨

Use this as a frontend/design reference for Copilot. No backend logic. Focus on look, feel, layout and reusable styling patterns.

Visual direction

  • Playful, vibrant and full of energy.
  • Word cloud is the hero element.
  • One big main word in the center.
  • Many surrounding words in different sizes, colors, rotations and fonts.
  • Organic cloud shape, not linear or aligned.
  • Words overlap, float and cluster naturally.
  • Rich gradients, soft glows, shadows and depth.
  • Floating particles, sparkles, blobs and confetti.

Overall feeling

A joyful, immersive, magical creativity experience. Make it feel like an explosion of ideas and collaboration.

Use this file as style inspiration, not final production code.
