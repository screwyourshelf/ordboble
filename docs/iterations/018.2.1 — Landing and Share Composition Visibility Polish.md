# Iteration 018.2.1 — Landing and Share Composition Visibility Polish

## Goal

Improve readability and visual composition on the landing/share experience.

Important text and word-cloud words are currently being visually washed out by decorative gradients and glow layers.

This iteration should improve:
- hero title readability
- central word readability
- left/right composition balance
- glow containment
- large-screen visual polish

without redesigning the landing page.

---

# Problem

## 1. Hero title readability

The large `Ordboble` hero title can become partially hidden or washed out because the right-side word-cloud glow bleeds too far into the left hero content column.

The text is technically rendered, but visually loses contrast.

---

## 2. Word cloud center readability

Some words in the word cloud become hard to read when placed over the brightest center glow.

The center glow currently competes with the words.

Words should remain readable first.

Glow should support the composition, not obscure it.

---

## 3. Left/right composition separation

The boundary between:
- left hero copy
- right visual word cloud

is too soft.

The right-side visual atmosphere invades the left reading area.

The goal is not a hard divider, but better visual protection for readable content.

---

# Scope

## In Scope

- Reduce glow bleed into text-heavy areas
- Improve `Ordboble` hero title readability
- Improve readability of central word-cloud words
- Refine center glow intensity
- Improve left/right composition balance
- Improve large-screen/ultrawide layout behavior
- Keep atmosphere and brand feeling intact
- Make small targeted visual changes only

## Out of Scope

Do NOT:
- redesign the landing page
- redesign the word cloud system
- remove the colorful/glowy identity
- add new features
- add new dependencies
- add a new layout engine
- change backend/realtime/session logic
- change routing
- change product flow

---

# Visual Direction

The page should still feel:
- playful
- atmospheric
- colorful
- alive
- immersive

But it should become:
- more readable
- more balanced
- less visually noisy
- more premium
- more intentional

---

# Required Improvements

## Left column protection

Improve readability of the hero content area.

Possible approaches:
- constrain the right-side cloud container
- reduce leftward glow bleed
- add subtle fade/mask between columns
- reduce orb radius near the content boundary
- increase breathing room between content and cloud
- improve z-index/layering if needed

Avoid:
- hard vertical divider
- flat/corporate layout
- killing the glow identity

---

## Center glow refinement

The brightest glow in the word cloud should be reduced.

Possible approaches:
- lower center glow opacity
- reduce blur radius
- add subtle darkening/vignette behind words
- move hotspot slightly away from central words
- improve contrast behind text

Avoid:
- heavy outlines
- harsh text shadows
- ugly strokes
- removing glow entirely

---

## Word readability

Words near bright glow areas should remain readable.

Possible approaches:
- tune word text-shadow/drop-shadow
- reduce background glow behind central words
- adjust word layering/depth
- adjust opacity of decorative light behind words

---

## Large-screen behavior

Check composition on wide desktop screens.

Improve:
- max-width behavior
- cloud scale
- decorative layer spread
- large-screen spacing
- left/right visual balance

---
