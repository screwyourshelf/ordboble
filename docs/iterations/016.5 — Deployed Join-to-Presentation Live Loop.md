# Iteration 016.5 — Deployed Join-to-Presentation Live Loop

## Goal

Verify and harden the complete deployed realtime collaboration loop.

This iteration validates that the deployed Ordboble system works across real browser sessions and devices:

```text
Host creates session
→ share page generates QR/join URL
→ contributor joins /join/:id
→ contributor submits words
→ Worker stores words in D1
→ SSE emits updates
→ /present/:id updates live
```

This is the first full deployed collaborative MVP verification.
