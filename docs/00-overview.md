# Ordboble

Ordboble is a playful collaborative word cloud web app.

A host creates a word cloud, shares a link or QR code, and contributors submit words live from their phones.

The word cloud updates in realtime and can later be:

* customized
* presented fullscreen
* downloaded
* printed
* shared

---

# Main Goals

* extremely simple to use
* visually delightful
* lightweight and fast
* mobile-first for contributors
* beautiful on large presentation screens
* playful but polished

---

# Main Roles

## Host

The host can:

* create a cloud
* configure settings
* share QR/link
* present the cloud
* customize themes/shapes
* export and print

## Contributor

The contributor can:

* open link or scan QR
* submit words quickly from mobile
* see friendly confirmation feedback

---

# Visual Direction

The app should feel:

* colorful
* alive
* expressive
* collaborative
* modern
* joyful

Inspired by:

* Canva
* Mentimeter
* tldraw
